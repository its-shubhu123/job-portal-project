# Job Portal (Mini LinkedIn)
Full-stack job portal built with the MERN stack featuring JWT authentication, role-based access, job listings, and application management.

# Features
- Secure authentication using JWT for job seekers and employers
- Browse and search job listings
- Apply for jobs and manage applications
- Recruiter dashboard to manage job postings
- Responsive design for all devices
  
# Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose
- Authentication: JWT, bcrypt

# Project Status
MVP completed. Actively improving features and code quality.
