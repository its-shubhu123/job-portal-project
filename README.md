# Job Portal (Mini LinkedIn)

A full-stack **Job Portal** built using the **MERN Stack**, featuring secure authentication, role-based access, job listings, and application management.

---

## Features
-  Secure authentication using **JWT**
-  Role-based access for **Job Seekers & Employers**
-  Browse and search job listings
-  Apply for jobs and manage applications
-  Recruiter dashboard to manage job postings
-  Fully responsive design

---

##  Tech Stack

### Frontend
- React.js  
- HTML, CSS, JavaScript  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  
- Mongoose  

### Authentication
- JWT  
- bcrypt
